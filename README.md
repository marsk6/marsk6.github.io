# Marsk's Blog
My blog is built with react, nextjs, keystone.

# Dev
Blog admin uses keystone, a headless cms, to manage blog data.

run blog admin
```shell
git clone git@github.com:marsk6/blog-admin.git
```

```shell
yarn keystone:dev
```
visit admin: localhost:3000

run blog site
```shell
yarn
yarn next:dev
```

visit blog: localhost:4000
